# Semantic-Segmentation
Using a U-net for semantic segmentation

**Dataset**
The dataset should have the following directories
|train_set (\n)
    |--original_imagees
    |--masks
|test_set
    |--original_images
    |--masks
|val_set
    |--original_images
    |--masks
